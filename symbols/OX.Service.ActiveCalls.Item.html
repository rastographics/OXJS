(void) hangup(callbacks)

The XMPP-API requires one of two commands to be called to end a call based on whether or not the call has been answered (confirmed). This is a convenience funtion to make the correct API call based upon the dialog state of the current this object.
Parameters:
(Object) callbacks Optional
An object supplying functions for 'onSuccess' and 'onError'.
(Function) callbacks.onSuccess Optional
The success callback.
(Function) callbacks.onError Optional
The error callback.
