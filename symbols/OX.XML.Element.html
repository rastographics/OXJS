Overview

A simple XML element class.

Extends:
OX.Base

  var newElement = OX.XML.Element.extend({name: 'foo'});
  newElement.attr('bar', 'bam');
  newElement.addChild(OX.XML.Element.extend({name: 'child'}));
  newElement.toString();
  // -> '<foo bar="bam"><child></child></foo>'
