(String) action()

Return the action, if any, in the query parameters.
Returns:
(String) The action of this query, or undefined if none found.

  var uri = OX.URI.parse('xmpp:lisa@example.com?message');
  uri.action();

(String) queryParam(param)

Return the value, if any, for a parameter in the query parameters.
Parameters:
(String) param
The parameter who's value is looked up.
Returns:
(String) The value of the parameter, or undefined if not found.

  var uri = OX.URI.parse('xmpp:lisa@example.com?;item=1');
  uri.queryParam('item');
  // -> '1'

(String) toString()

Convert URI object to string representation.
Returns:
(String) The URI converted to a string.

  OX.URI.extend({ scheme: 'sip',
                  path: 'lisa@example.com' }).toString();
  //-> "sip:lisa@example.com"
